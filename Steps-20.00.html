13) Fix the Title of the Visuals. 20

After we are done building the report, now we need to the title for all the visuals.
It's easier to format them at the end because we will be moving from one to another and applying the same format.


  1. Select the visual.
  2. Select format.
  3. Select title
  4. Change the    Font color:    White           Background Color:   #0E1A77               Alignment : Center
  5. Select the next visual and apply the same steps.
