8) Create the KPIs. 15

Now it is  time to create the KPIs, These are the steps to follow.

STEP 1: Create Cost Per Claim card  and YOY  card.

  1. Double click on the Card Icon On the visualization bar.
  2. Card visual will display on the report
  3. Drag an drop the ".Cost per Claim" column from the TSA claim table.
  4. Do the same 1 to 3 process to create YOY Card.



STEP 2: Configure the "Cost Per Claim" Visual

  1. Click on the Format icon.
  2. Fix the  Card  Size:     Weight :   240                    Height: 127.
  3. Fix the Data Label :     Text size :  30.
  4. Turn Off Category .
  5. Turn on the title .
  6. Type "Cost Per Claim".
  7. Turn on the border color :    Color  light grey      radius:  10.
  8. Select the "Cost Per Column" Column .
  9. Change the Format to:       Currency                  select the "," sign .


STEP 3: Configure the "YOY" Visual

  1. Click on the Format icon
  2. Fix the  Card  Size:     Weight :   240                    Height: 127.
  3. Fix the Data Label :     Text size :  30.
  4. Turn Off Category .
  5. Turn on the title .
  6. Type "YOY".
  7. Turn on the border color :    Color  light grey      radius:  10.
  8. Select the "YOY" Column .
  9. Change the Format to:       Percentage .
